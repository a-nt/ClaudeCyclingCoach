## Features

✓ Activity analysis (power/HR coupling, zones, decoupling)
✓ Fitness trends (CTL/ATL/TSB progression)
✓ Athlete profile management
✓ Conversational coaching follow-ups
✓ Secure credential management
✓ Zero external dependencies (Node.js built-ins only)
✓ Distributable to any intervals.icu user

## Commands

/coach setup          - Configure credentials
/coach profile        - View athlete profile
/coach analyze [id]   - Analyze activity
/coach trends [days]  - View fitness trends
/coach                - General coaching

## Requirements

- Node.js (any recent version)
- intervals.icu account with API key
- Claude Code

## Installation

See README.md for detailed instructions.

Quick start:
1. Copy directory to ~/.claude/skills/cycling-coach/
2. Run /coach setup in Claude Code
3. Enter your intervals.icu credentials
4. Start analyzing!
